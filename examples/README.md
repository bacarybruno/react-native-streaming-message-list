# Examples

This folder contains example implementations of `react-native-streaming-message-list`.

## Running Examples

1. Set the demo variant in `.env` at the project root:

```bash
# For ChatGPT-style demo
EXPO_PUBLIC_DEMO_VARIANT=chatgpt

# For basic demo
EXPO_PUBLIC_DEMO_VARIANT=basic
```

2. Run from the project root:

```bash
# iOS
yarn example ios

# Android
yarn example android

# Web
yarn example web
```
